# Weather-Indicator
날씨를 출력하는 날씨 표시기 만들기

<br />

## 날씨 API
Open weather API: https://openweathermap.org/

<br />

## 필수 기능
- [x] 지역별 날씨
- [x] 날씨 이미지/이모지 포함

<br />
